### 2. 5색 컬러 팔레트 런타임 전환 시스템

#### 팔레트 구성
| # | 이름 | brand-800 | 용도 |
|---|---|---|---|
| 1 | Ocean Blue | `#1E3A8A` | 기본 (다크블루·로열블루) |
| 2 | Emerald Teal | `#115e59` | 청록 계열 |
| 3 | Royal Purple | `#5b21b6` | 보라 계열 |
| 4 | Forest Green | `#166534` | 녹색 계열 |
| 5 | Rose Red | `#9f1239` | 빨강 계열 |

#### 구현 방식
- `tailwind.config.js`: brand 색상을 CSS 변수(`var(--brand-*)`)로 변경
- `index.css`: `html.palette-N` 선택자로 5개 팔레트 변수 정의
- `ThemeContext.jsx`: `palette` 상태 + localStorage 저장 + FOUC 방지 스크립트
- **헤더 우측 5개 컬러 원**: 클릭 즉시 전체 색상 전환

#### 화이트 베이스 원칙
- **배경**: 라이트 모드 = 흰색, 다크 모드 = 중립 zinc (팔레트 무관)
- **액센트**: 제목 텍스트·버튼·아이콘·링크만 팔레트 색상 적용
- Hero/배너 다크 섹션: `bg-neutral-900` 고정 + 팔레트 색 미묘한 글로우(opacity 20%)

### 4. 헤더 드롭다운 재설계

**문제**: 마우스를 드롭다운 패널로 이동 시 메뉴 닫힘  
**원인**: 기존 메가메뉴 방식에서 nav 요소 gap 발생  
**해결**: "강의 영상" `<li>` 내부에 `absolute` 드롭다운 포함

```
<li onMouseEnter/onMouseLeave>   ← li 밖으로 나갈 때만 닫힘
  <NavLink>강의 영상</NavLink>
  <VideoDropdown />              ← li 안에 포함, gap 없음
</li>
```

- 6개 주제 2열 그리드 + 아이콘 + 강의 수 표시
- 하단 "전체 강의 영상 보기" 링크

### 5. Hero 배경 개선

**문제**: `Placeholder` 컴포넌트의 "HERO 01 / auto" 텍스트 노출  
**해결**: CSS 그라디언트 + 격자 패턴 + 팔레트 컬러 글로우로 교체

```
bg-neutral-900 (중립 다크 고정)
+ radial-gradient(var(--brand-700), opacity 22%)  ← 팔레트 힌트
+ 격자 패턴 (rgba 2%)
+ 하단 페이드 그라디언트
```

실제 배경 이미지 사용 시: `slides` 배열의 `bg: null` → 이미지 URL 입력
