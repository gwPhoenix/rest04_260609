## 변경 내역

### 1. 홈 중복 섹션 3개 → 1개 통합

**문제**: 같은 기능(주제별 강의 이동)을 하는 섹션이 세 개 연달아 배치
| 기존 섹션 | 내용 |
|---|---|
| TopicLinks | "AI 기초부터 딥러닝까지..." + 알약 버튼 6개 |
| OurVideos | "AI 강의 영상 / 주제별로 배우세요" + 카드 6개 |
| FeaturedBand | "엄선된 AI 강의로 오늘 바로 시작하세요" 다크 배너 |

**해결**: `TopicsSection` 하나로 통합
```
[섹션 제목] AI 강의 영상 / 주제별로 배우세요
[소개 문구] AI 기초부터 딥러닝까지, 누구나 쉽게 배울 수 있습니다.
[6개 카드 그리드] 3×2 (데스크탑) / 2×3 (모바일)
[구분선 + CTA] 엄선된 AI 강의로 오늘 바로 시작하세요 | 전체 강의 보기 →
```

**홈 최종 구조**: Hero → TopicsSection → LatestVideos (3단 구성)

---

### 2. 강의 영상 드롭다운 메뉴 재설계

**문제**: 헤더 "강의 영상" 메뉴 → 드롭다운으로 마우스 이동 시 즉시 닫힘  
**원인**: 기존 메가메뉴 방식에서 nav 경계 이탈 시 hovered 상태 초기화  
**해결**: 드롭다운을 해당 `<li>` 내부에 `absolute` 배치

```jsx
// 이전: nav 전체 onMouseLeave → 모든 항목 닫힘
// 이후: li onMouseLeave → 해당 li 범위 벗어날 때만 닫힘
<li onMouseEnter={() => setVideoOpen(true)}
    onMouseLeave={() => setVideoOpen(false)}>
  <NavLink>강의 영상</NavLink>
  {videoOpen && <VideoDropdown />}  {/* li 안에 포함 */}
</li>
```

드롭다운 UI: 6개 주제 2열 그리드 + 아이콘 + 강의 수 + "전체 보기" 링크

### 3. Hero 플레이스홀더 제거

**문제**: `Placeholder` 컴포넌트가 "HERO 01 / auto" 텍스트를 노출  
**해결**: CSS 그라디언트 + 격자 패턴 배경으로 대체

```jsx
// 배경 구성
bg-neutral-900 (항상 중립 어두운 배경)
+ radial-gradient(var(--brand-700), opacity 22%)  // 팔레트 힌트 글로우
+ 격자 선 패턴 (opacity 2%)
+ 하단 페이드 그라디언트
```

실제 이미지 적용 시: `slides` 배열의 `bg: null` → 이미지 경로 입력

### 4. 화이트 베이스 디자인 완성

**기조**: rest03처럼 배경은 항상 흰색, 팔레트는 텍스트·버튼·아이콘만 변경

| 영역 | 라이트 모드 | 다크 모드 |
|---|---|---|
| 본문 섹션 | `bg-white` | `bg-zinc-900` |
| 히어로/다크 섹션 | `bg-neutral-900` | `bg-neutral-900` |
| 팔레트 효과 | 제목·버튼·아이콘 | 제목·버튼·아이콘 |

---

### 5. 다크 모드 배경색 중립 zinc 통일

**문제**: 팔레트별 `--dark-bg` 값이 다름 → 파란/녹색/빨간 배경 발생
```
palette-1: --dark-bg: #080E1E  (남색)
palette-2: --dark-bg: #021916  (진녹색)
palette-5: --dark-bg: #1a0008  (진빨간색)
```

**해결**: 모든 팔레트 공통으로 중립 zinc 고정
```css
/* index.css — 팔레트 오버라이드 차단 (동일 명시도, 뒤에 선언) */
html.palette-1, html.palette-2, html.palette-3, html.palette-4, html.palette-5 {
  --dark-bg:      #09090b;  /* zinc-950 */
  --dark-surface: #18181b;  /* zinc-900 */
  --dark-card:    #27272a;  /* zinc-800 */
  --dark-border:  #3f3f46;  /* zinc-700 */
}
```
