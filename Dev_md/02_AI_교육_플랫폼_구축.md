# 02. AI 온라인 교육 플랫폼 구축

작업일: 2026-06-09

## 목표
rest03(건설사 템플릿)을 기반으로 **AI 관련 유튜브 동영상 콘텐츠를 제공하는 온라인 교육 회사 사이트**를 새로 구축한다.

## 요구사항 요약
- 주 컬러: 다크블루 + 로열블루, 보조/포인트 컬러 포함 5색 팔레트
- 다크 모드 & 라이트 모드 전환 기능
- 모바일 최적화 (반응형)
- AI 기초·AI 리터러시 등 주제별 유튜브 영상 2열×3행 그리드 + 페이지네이션
- 주제별 영상 메뉴 분류

---

## 기술 선택
| 항목 | 내용 |
|---|---|
| 스택 | React 18 + Vite 5 + Tailwind CSS 3 + React Router v6 |
| 라우팅 | HashRouter (GitHub Pages 정적 호스팅 호환) |
| 다크모드 | Tailwind `darkMode: 'class'` + ThemeContext (localStorage 저장, FOUC 방지 스크립트 포함) |
| 영상 임베드 | YouTube facade 패턴 (썸네일 → 클릭 시 `youtube-nocookie.com` iframe 전환) |

## 라우트 구조
```
/                     → 홈
/videos/:topic        → 강의 영상 (ai-basics, ai-literacy, machine-learning, deep-learning, tutorials, trends)
/curriculum           → 커리큘럼
/about                → 회사 소개
/contact              → 문의하기
/privacy              → 개인정보처리방침
/terms                → 이용약관
```

## 영상 데이터 구조
- 6개 주제 × 8개 영상 = **총 48개** 영상 슬롯
- 각 영상: `youtubeId` (null → 실제 ID 교체), `title`, `desc`, `duration`, `date`, `level`
- `youtubeId: null` 상태에서는 플레이스홀더 썸네일 표시
- ⚠️ YouTube '비공개' 영상은 임베드 불가 — '공개' 또는 '링크 공개' 상태 필요

## 영상 그리드 (Videos.jsx)
- **2열 × 3행 = 6개/페이지** (모바일: 1열)
- 주제 탭 sticky (헤더 바로 아래)
- 페이지네이션: 이전/다음 + 페이지 번호
